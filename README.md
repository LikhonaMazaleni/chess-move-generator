# chess Move Generator

This project is a Python program that generates valid chess moves based on given rules.

I worked on this project to improve my problem solving skills and to better understand how to break down complex problems into smaller steps.

Through this project, I practiced:
- Logical thinking
- Working with conditions
- Structuring code clearly

It helped me realise how important it is to think through a problem before writing code.

This project is part of my learning journey and I plan to improve it over time.
